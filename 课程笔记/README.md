# 课程笔记

这个目录下是各个课程的笔记，先是**按照必选修**分类，在必修或选修文件夹下**按照课程类型**进行分类，每个课程类型下有**各个老师**的课程子文件夹（老师名称**后跟年份**，你可以快速确定课程笔记是否有最新内容）（如果该课程只有一个老师的文件夹，说明学长没有找到其他老师的课程，希望你能帮忙提交PR，但整体上学校的课程都大差不差）。  


## 目录结构
你可以使用该目录快速定位到你需要的课程笔记，该目录**只索引到老师文件夹下** ，具体笔记请到对应文件夹下查找。  

例：  

```
课程笔记
├── 必修
│   ├── 高等数学1
│   │   ├── [杜毅2025](https://github.com/vacuoleC/JNU_LST_BS/tree/main/课程笔记/大一/高等数学1/杜毅2025)
│   ├── 英语
│   │   ├── 李志军2025
│   │   ├── 谭岸青2025
│   └── ……
├── 
│   ├── ……
├── ……
```
